# Weather App
🌐 **Live Demo:** [Weather App](https://playful-axolotl-345743.netlify.app/)

A modern weather application built with React, Vite, and TailwindCSS. This app provides real-time weather updates using a sleek and responsive user interface.

- [Features](#-features)
- [Installation](#-installation)
- [Usage](#-usage)

## ✨ Features

- 🌤️ Real-time weather updates
- 🎨 Styled with TailwindCSS for a modern UI
- ⚡ Lightning-fast performance with Vite
- 🌀 Fully responsive design
- 🔥 React Hooks for state management

## 🚀 Installation

Make sure you have **Node.js** installed, then follow these steps:

```sh
# Clone the repository
git clone https://github.com/yourusername/weather-app.git
cd weather-app

# Install dependencies
npm install
```

## ▶️ Usage

Start the development server:

```sh
npm run dev
```

Build for production:

```sh
npm run build
```

Preview the built app:

```sh
npm run preview
```

Lint the project:

```sh
npm run lint
```

## 📜 Scripts

The following scripts are available in the project:

- `dev` → Starts the development server using Vite
- `build` → Builds the project for production
- `preview` → Previews the built app
- `lint` → Runs ESLint to check code quality
